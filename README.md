# 🧑‍💻 Data Structures and Algorithms in Java  

This repository contains my implementation of **Data Structures and Algorithms (DSA)** in **Java**.  
The goal of this project is to strengthen my problem-solving skills and build a strong foundation in **core computer science concepts**.  

---

## 📂 Contents  

- **Arrays & Strings**  
  - Searching, Sorting, Prefix Sum, Sliding Window, etc.  

- **Linked List**  
  - Singly, Doubly, Circular Linked List  
  - Operations: Insert, Delete, Reverse  

- **Stack & Queue**  
  - Stack using Array & Linked List  
  - Queue, Circular Queue, Deque, Priority Queue  

- **Recursion & Backtracking**  
  - Factorials, Fibonacci, N-Queens, Rat in Maze  

- **Trees**  
  - Binary Tree, Binary Search Tree  
  - Traversals (Inorder, Preorder, Postorder, Level Order)  
  - Lowest Common Ancestor, Diameter of Tree  

- **Graphs**  
  - BFS, DFS  
  - Topological Sort, Cycle Detection  
  - Dijkstra’s Algorithm, Kruskal’s, Prim’s  

- **Dynamic Programming (Dp47part2.DP)**  
  - Fibonacci, Knapsack, LIS, LCS, Matrix Chain Multiplication  

- **Greedy Algorithms**  
  - Activity Selection, Huffman Coding  

- **Sorting Algorithms**  
  - Bubble, Insertion, Selection, Merge Sort, Quick Sort, Heap Sort  

- **Miscellaneous**  
  - Hashing, Bit Manipulation, Sliding Window, Two Pointer
